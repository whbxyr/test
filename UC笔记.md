### 前端相关技术
+ **scrat**
   github地址: [https://github.com/scrat-team/scrat](https://github.com/scrat-team/scrat)
   安装方法：
```bash
$ npm install -g scrat
```
实例下载并执行，执行以下命令后预览项目：在浏览器中打开链接[http://127.0.0.1:5000](http://127.0.0.1:5000)：
```bash
$ git clone https://github.com/scrat-team/scrat-site.git
$ cd scrat-site
$ scrat install
$ scrat release
$ scrat server start
```
